# 微信小程序端 和 网页端 的区别
## 网页端开发三大标准

结构 ：HTML               -     div, span, h1, p      标签

样式： CSS					-     .box{ color:#fff; }    选择器{ 样式 }

行为： JS                       -      ECMAScript (语法)   +     BOM(浏览器对象模型)    +     DOM(文档对象模型)  

## 微信小程序开发四大标准

ML 标记语言 markup language

结构 ：`WXML`                        -   `<view>`    `<text>`    ，不要用 `html` 标签

样式： `WXSS`                        -     CSS  +  微信小程序独特的功能

行为： `JS`                             -     ECMAScript (语法)   +    WX 对象(调用微信功能)

配置：`JSON`						-    小程序项目配置
